3o|||sheets IDE

A lightweight, cross-platform IDE built on OpenJDK. Runs even on single-board computers with minimal resources.

Supports LD, FBD, and ST languages plus a virtual machine instruction set for advanced logic design.

In debug mode, developers can pause and rewrite individual tasks live — without stopping the system.

  • Classic PLC mode — cyclic model (OB, OB100).
  • RTOS mode — preemptive multitasking via 1 ms timer interrupt.

3osheet Compiler

The 3o|||sheet Compiler compiles instruction text into bytecode.

Its .3osheet language blends high-level syntax with atomic assembly-like instructions executed by the VM runtime.

Supports virtual stack and context management — ideal for recursive multitasking programs.

3osheet Runtime

Register-based Virtual Machine

A register-based virtual machine running safely in a sandbox. Works on devices with only 8 KB RAM and full peripheral access.

No architecture-specific code — portable to any MCU.

  • Runs on low-memory devices (~8 KB RAM)
  • Safe sandboxed execution
  • Controls GPIO, timers, UART, I²C, SPI, etc.
  • Architecture-independent design
