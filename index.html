3O|||SHEET is a lightweight development environment for programming industrial controllers LogicGather, with support for LD and ST programming languages according to the IEC 61131-3 standard.

3o|||sheets IDE

3o|||sheets IDE — a lightweight, cross-platform development environment built on OpenJDK. It runs even on single-board computers with minimal resources.

Supports LD, FBD, and ST languages, plus a low-level virtual machine instruction set for advanced logic design.

In debug mode, developers can pause and rewrite individual tasks live — without stopping the system.

  • Classic PLC mode — cyclic model (OB, OB100) for compatibility.
  • RTOS mode — preemptive multitasking via 1 ms timer interrupt, with full task context management.

Combines PLC simplicity with RTOS precision — designed for efficiency, portability, and real-time control.

3osheet Compiler

The 3o|||sheet Compiler is a standalone tool that compiles instruction text into bytecode.

Its custom .3osheet language combines high-level syntax for arrays and structures with assembly-like atomic instructions executed by the virtual machine.

It supports stack and context operations — allowing complex multitasking programs with recursion and deep function transitions.

3osheet Runtime

Register-based Virtual Machine

A register-based virtual machine executing code in a safe sandboxed environment. Runs on devices with only 8 KB RAM and provides full peripheral access.

No hardware-specific code — easily portable to any microcontroller.

  • Works on low-memory devices (~8 KB RAM)
  • Safe sandboxed execution
  • Controls GPIO, timers, UART, I²C, SPI, etc.
  • Architecture-independent implementation
